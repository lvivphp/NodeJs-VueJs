# NodeJs-VueJs
NodeJs+VueJs

git clone https://github.com/lvivphp/NodeJs-VueJs.git

create file .env

add lines
    ENV=prod
    DB_CONNECT=mongodb://localhost:27017/hotdog

npm install

npm install nodemon

npm run build

npm start


go to http://127.0.0.1:3000


Example https://nodejs-ua.herokuapp.com
